# temp_scipts

Small bioinformatics and utility scripts collected by topic.

## Directory Layout

```text
temp_scipts/
  getorganelle/
    get.py
    README.md
```

## Available Scripts

### getorganelle

Batch wrapper for GetOrganelle. It supports paired-end read discovery,
multi-sample assembly, resume protection, status reporting, cleanup, and final
`*.path_sequence.fasta` summary.

See [getorganelle/README.md](getorganelle/README.md) for usage.
